Busan Metro GTFS Feed

This project provides a General Transit Feed Specification (GTFS) dataset for the subway metro system in Busan, South Korea.

About the Project

The GTFS feed contains comprehensive transit data for the Busan Metro, including routes, stops, schedules, and other essential information. It is designed to support transit applications, journey planners, and research into public transportation in Busan.

Features

  • Detailed subway routes and schedules
  • Detailed stop locations and station information
  • Open data format compatible with most transit apps

Usage

Download the GTFS feed and integrate it into your transit application or analysis tool. The data is provided in the standard GTFS format.
